# Loon 代理图标

用于 Loon 策略组/节点的图标文件。

## 命名规范

参考 Qure 图标集命名：`<平台名>.png`（如 `Bybit.png`、`Binance.png`）

## 现有图标

| 图标 | 格式 | 来源 |
|------|------|------|
| Bybit | PNG / SVG | Tucsky/aggr |

## 图标来源

- [Qure 图标集](https://github.com/Koolson/Qure/tree/master/IconSet/Color) — 主流平台图标
- [Orz-3/mini](https://github.com/Orz-3/mini) — 迷你图标集

## 自定义图标

如果以上图标集没有你需要的平台：
1. 从平台官网下载 logo
2. 搜索引擎搜 `<平台> logo png`
3. 截图裁剪为正方形 PNG，放到本目录
